Grid Features
  • Virtualized rendering — handles millions of rows
  • Column resizing, reordering, pinning
  • Multi-column sort (click headers)
  • Cell selection + copy (Cmd+C copies as TSV)
  • Row selection with checkboxes
  • Column type icons (🔑 PK, 🔗 FK, etc.)
  • NULL displayed distinctly (italic gray)
Inline Editing
  • Double-click cell to edit in place
  • Pending changes highlighted (orange)
  • New rows highlighted (green)
  • Batch apply — review all changes before commit
  • Undo/redo per cell (Cmd+Z)
  • FK columns show resolved preview (id → name)
  • JSON/text columns expand in modal
Filtering
  • Smart filter bar (type SQL-like conditions)
  • Click column header → quick filter menu
  • Visual filter builder for non-SQL users
  • Save filter presets per table
  • Full-text search across all columns
Result Views
  • Grid view (default spreadsheet)
  • JSON view (formatted, collapsible)
  • Single-row detail view (form layout)
  • Chart view (auto-suggest chart type)
  • Quick export from any view

Key detail: FK columns (like user_id) show the foreign key value AND a preview of the referenced row. Click to navigate to that row in the related table.
