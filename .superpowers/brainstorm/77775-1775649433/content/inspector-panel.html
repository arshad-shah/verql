Inspector Tabs by Object Kind

Table / View

Info: schema, row count, size, PK, indexes, FKs, referenced-by
Columns: name, type, nullable, default, constraints
DDL: CREATE TABLE source
Deps: objects that depend on / are depended on

Function / Procedure

Info: language, return type, volatility
Params: name, type, IN/OUT/INOUT
Source: function body with syntax highlighting
DDL: CREATE FUNCTION source

Trigger

Info: event (INSERT/UPDATE/DELETE), timing (BEFORE/AFTER), table
Source: trigger body / function reference
DDL: CREATE TRIGGER source

Sequence / Enum / Extension

Sequence: current value, min/max, increment, cycle
Enum: list of values
Extension: version, schema, description

Behavior

• Panel appears on first object click, persists until closed via X button
• Clicking a different object updates the panel content (no re-animation)
• Panel width is resizable via drag handle
• Stored in UI store: inspectorVisible, inspectorWidth, selectedObject
