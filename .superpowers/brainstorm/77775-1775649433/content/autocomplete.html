Architecture

CompletionProvider — registered with Monaco, receives cursor position & text
• SQL Parser (lightweight) — determines context: SELECT clause? FROM clause? after dot? alias scope?
• Schema Resolver — reads from the cached schema store to resolve tables, columns, functions
• Completion Builder — produces Monaco CompletionItems with kind, detail, insertText, sortText

Data flow: Schema store (Zustand) → CompletionProvider reads cache → no IPC per keystroke
